WOCH STUDIOS SOUND BOARD APP v1.02

THIS PROJECT CAN BE VERY EASILY MADE INTO AN SOUND BOARD DESIRED BY FOLLOWING THE BELOW STEPS.

CREATING A SOUND BOARD
1. COPY SOUND FILES INTO THE DIR: SoundBoard\res\raw
2. ADD KEY-VALUE PAIRS TO SoundBoard\res\raw\filemap.txt
	A. THERE IS AN EXAMPLE KEY-VALUE PAIR IN THE FILE 
3.PACKAGE APP AND DEPLOY TO ANDROID DEVICE
